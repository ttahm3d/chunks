Introduction

Chunks is a component library with utility classes. Utility classes allow users to develop UI screens at a much faster speed than that is achieved when writing own css. There are few componenets which the user can create by just using few of the utilty classes. This allows the user to focus on creating the functionality rather than worrying about the design

To use these classes, all that one has to do is to add the below import in his / her css file. 

@import url("chunksui.netlify.app/css/style.css")
